# 识人簿

本地运行的人物观察与决策辅助工具。使用 Node.js、Express 和 SQLite，默认只监听 `127.0.0.1`。

## 启动

```bash
npm install
npm start
```

然后访问 `http://127.0.0.1:4173`。

数据保存在 `data/persona-notes.sqlite`。图片生成功能可选：复制 `.env.example` 中的环境变量并提供 API Key；未配置时接口会返回可编辑的安全提示词。

## 数据原则

- 尽量使用昵称并征得对方同意。
- 区分观察事实、本人表达和推测。
- 为性格结论关联具体观察证据和可信度。
- 自动保留人物档案修改历史，记录决策建议及事后反馈。
- MBTI 与星座只作为辅助标签，不作为事实或诊断。
- 支持完整备份、单人删除和全部数据删除。

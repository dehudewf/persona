# Project Skills

This folder contains local skills for the personality analysis and character image generation website project.

Installed skills:

- `frontend-design-anthropic`: distinctive frontend interface design
- `frontend-design-product`: product-focused frontend pages and flows
- `ui-ux-pro-max`: UX flow, design system, and implementation planning
- `soul-md-maker`: structured personality and persona profile generation

Use these skills when designing the website, building frontend screens, planning the user journey, or generating structured persona data for character image prompts.

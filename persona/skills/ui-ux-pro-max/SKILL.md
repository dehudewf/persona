# UI/UX Pro Max

Use this skill to turn a feature idea into a complete user experience and design system.

## UX Workflow

1. Map the user's path from entry to first useful result.
2. Identify required screens, decisions, and system states.
3. Define components and reusable patterns.
4. Specify interaction details, validation, accessibility, and responsive behavior.
5. Convert the UX into file-level implementation tasks.

## Recommended Flow For This Project

1. Landing or app home: explain "import personality material, get a creative persona and visual direction."
2. Input: paste text, upload notes, or answer guided prompts.
3. Consent and framing: clarify that outputs are creative/personality insights, not clinical evaluation.
4. Analysis loading: show progress steps such as extracting traits, building archetype, preparing visual direction.
5. Result: show personality summary, archetype, trait map, image prompt, and generated character concept.
6. Refinement: let users adjust tone, era, style, color mood, role, and symbolic elements.
7. Save/share/export: save persona profile, copy prompt, or export result.

## Component Checklist

- Input composer
- Prompt/question wizard
- Privacy and framing notice
- Trait summary panel
- Archetype badge
- Character concept card
- Style controls
- Regenerate button
- Saved personas list
- Compare mode
- Empty, loading, error, and success states

## Design System Checklist

Define tokens for color, type, spacing, radius, shadow, focus, and motion. Include contrast-safe colors and keyboard-visible focus states. Keep forms clear and mobile-friendly.

## Product Safety

Avoid diagnostic language. Prefer phrases such as "creative interpretation", "possible pattern", "persona direction", and "story/visual concept".
